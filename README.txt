Lunch Jackpot

Upload all files to the root of your GitHub repository.

Features:
- Slot-machine lunch picker
- Spin history saved in the browser
- Statistics saved in the browser
- Confetti for Lucky, Legendary, and Chef's Surprise
- Sound on/off toggle
- QR-friendly mobile website, no install required

To update foods:
Edit foods.json, commit to GitHub, and GitHub Pages will update automatically.
